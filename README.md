React counter component 

>uses a state to store the initial value 0
> the component renders a jsx code
>with heading and state variable as argument
>and they contain 2 buttons
>named increase and decreces which take a onClick event
>to store a function updater
>so when the component is rendered initially the value will be zero
>as mentioned earlier there exist 2 button to increase and decrease
>so when clicking on increase button the value increase to initial state + one
>similarly when clicking on decrease button the value decrease to initial state - one
